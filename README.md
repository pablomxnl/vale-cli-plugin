# Vale CLI Plugin
> [!TIP]
> Pet project. Not associated with Errata (Vale CLI creators) or JetBrains whatsoever

<!-- Plugin description -->
        <p>
            Plugin that uses <em><a href="https://vale.sh/">Vale CLI</a></em> to check markdown,asciidoc and other documentation formats
        </p>

<!-- Plugin description end -->

## Installation

- Using the IDE built-in plugin system:
  
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>Marketplace</kbd> > <kbd>Search for "Vale CLI"</kbd> >
  <kbd>Install</kbd>
  
- Using JetBrains Marketplace:

  Go to [JetBrains Marketplace](https://plugins.jetbrains.com/plugin/19613-vale-cli) and install it by clicking the <kbd>Install to ...</kbd> button in case your IDE is running.

  You can also download the [latest release](https://plugins.jetbrains.com/plugin/19613-vale-cli/versions) from JetBrains Marketplace and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>

- Manually:

  Download the [latest release](https://gitlab.com/pablomxnl/vale-cli-plugin/-/releases/permalink/latest) and install it manually using
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
